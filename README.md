# Mansour Cup 2026 (Public Site)

## فكرة الموقع
موقع جماهيري ثابت (Static) يُنشر على GitHub Pages.
يعرض:
- ترتيب كل مجموعة (A/B/C/D)
- نتائج مباريات كل مجموعة
- صفحة تفاصيل لكل مباراة (حكم/معلق/أفضل لاعب + قائمة الهدافين داخل المباراة إن أدخلتها)

## كيف يتم تحديث النتائج (أسهل طريقة بدون تخريب ملفك الأصلي)
1) أنت تستمر في الإدخال داخل ملف Excel الأصلي (XLSM) كما هو.
2) بعد كل يوم مباريات: صدِّر/احفظ ملف CSV باسم:
   `data/matches.csv`
   بنفس الأعمدة الموجودة في هذا الملف.
3) ارفع التعديل إلى GitHub (Commit/Push).
4) الجمهور سيشاهد التحديث تلقائياً عند تحديث الصفحة (قد يحتاج دقائق بسبب كاش GitHub Pages).

> مهم: الموقع لا يقرأ ملف Excel مباشرة من GitHub Pages. GitHub Pages لا يشغّل Excel ولا ماكرو.
> نحن نستخدم CSV كجسر آمن وبسيط.

## أعمدة matches.csv
- group: A/B/C/D
- round: الجولة الأولى/الثانية/الثالثة...
- date: YYYY-MM-DD
- time: HH:MM
- team1, team2
- score1, score2 (اتركها فارغة قبل اللعب)
- referee1, referee2
- commentator
- player_of_match
- goals_team1, goals_team2
  - اكتب الهدافين بهذا الشكل: `اسم لاعب (2); اسم لاعب (1)`
  - إذا لا تريد تفاصيل الهدافين اتركها فارغة.
- match_code: معرف فريد للمباراة (A01..)
